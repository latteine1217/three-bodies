# STATUS

版本：**v1.0**（第一版正式系統）
更新日期：2026-06-13
狀態：功能完整、可執行（瀏覽器即時模擬，無建置流程）。

## 已實作

### 物理（`physics.js`，零 Three.js 依賴）
- PEFRL 四階辛積分；每幀 32 子步。純重力能量漂移 ~0%。
- Plummer 軟化重力（`U=−Gm₁m₂/√(r²+ε²)`，`ε=ras=0.15`），力與位能一致。
- 三星不等質量、非共面 3D 初始組態（半徑 ~4 含隨機）+ 隨機初速（spin=0 也不死寂）。
- 統一邊界收束：所有天體（含行星）共用「彈簧回復力 + 向外阻尼」邊界 `containR=8`；`DD=20` 解體安全網。
- 行星與星體同範圍漫遊（0 ~ containR）。

### 渲染（`renderer.js` / `stars.js` / `galaxy.js` / `background.js` / `blackbody.js`）
- 電漿恆星 shader：米粒組織、黑子、邊緣昏暗（limb darkening）。
- 黑體輻射星色（質量→色溫，放寬以加大變化）。
- 星芒眩光（緊湊核 + 柔和日冕 + 繞射尖刺），sine 緩慢閃爍。
- 程序烘焙 4096×2048 銀河貼圖，貼於緩慢自轉天球；視差星場 + 遠方裝飾星。
- 後製：UnrealBloom（圓形）→ OutputPass(ACES) → film grain + 暗角 + 色差（攝影質感）。

### 玩法（`civilization.js` / `ui.js`，《三体》主題）
- 行星溫度含大氣／水體熱慣性緩衝（`THERMAL_TAU`）。
- 恆／亂紀元（滑動視窗 + 遲滯緩衝）。
- 文明：毀滅後孵化（`LIFE_TAU`）→ 指數加速成長（`GROWTH_ACCEL`）→ 六層級。
- 五種災難：引力撕裂（洛希極限）、巨日、三日凌空、三飛星、黑暗森林打擊、系統崩潰。
- 經典毀滅宣告、文明日誌、歷代排行榜、紀元鐘。
- Sci-fi HUD（切角發光、掃描光條、可開合面板）+《三体》名言輪播。

## 已知限制

- 邊界收束的阻尼為刻意外部控制，啟用時不嚴格守恆能量／動量（純重力本身守恆 ~0%）。
- 溫度模型含一個經驗校準常數（0.7），使居中行星的均溫落回宜居帶。
- 效能瓶頸在渲染（per-pixel bloom/背景）；計算端可忽略。
- 無自動化測試套件；驗證以 Node 純模組基準 + 瀏覽器目視為主。
